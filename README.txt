To launch Mcalibrator on MATLAB, please run
>> Mcalibrator2

If you face some problems related to "tabselectionfcn" function, please run
>> Mcalibrator2_tabfunc_disabled
instead of Mcalibrator2.

We are happy if Mcalibrator2 can help your research projects. Thank you.
